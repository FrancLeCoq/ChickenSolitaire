# 🃏 Chicken Solitaire

Klondike Solitaire featuring **Francis le Coq**, in the same visual universe as the other
Francis le Coq mini-games (purple/gold theme, `$FRANC` / ⭐ holder unlocking, FR/EN).

## Play

Open `index.html` (single-file web app, designed as a Telegram Web App / GitHub Pages page).

## Menu

- **Play** — choose a game mode.
- **Connect Wallet** — links a wallet and detects `$FRANC` / ⭐ status.
- Unlock note + language switch (🇫🇷 / 🇬🇧).

## Game modes

| Mode | Access | Draw |
|------|--------|------|
| **Play – 1 carte** | Everyone | 1 card from the stock |
| **Play – 3 cartes** | `$FRANC` holders or ⭐ only (padlock otherwise) | 3 cards |

Each mode shows its **best score** and **best time**.

## Hints (Indices)

Three hints per game. Hints are **reserved for holders / ⭐** — in *Play – 1 carte* a
non-holder sees an orange padlock on the **Indice** button; tapping it opens the unlock modal.

## Rules

Standard Klondike:
- Build the four foundations up by suit from **A → K**.
- Build the tableau down in **alternating colours**; only **Kings** go on empty columns.
- **Tap** a card to select it, tap a destination to move; **double-tap** to auto-send to a foundation.
- Tap the stock to draw / recycle.

Win all four foundations → a **festive win screen** with Francis and fireworks 🎆.

## Assets (`assets/`)

Rooster artwork by suit (`pic`, `coeur`, `carreau`, `trefles`), face cards
(`roi`, `reine`, `vallet`), the mascot `francis`, the sad `triste` and the festive `festif`.
